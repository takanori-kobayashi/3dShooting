Unityプロジェクト(2022.3.16f1)

**3Dシューティング**  
自機を操作し最終地点のボスを倒すのが目的のゲームです。

|操作         |キーボード           |ゲームパッド|
|-------------|---------------------|---------- |
|上昇         |S　↑キー	            |方向キー下  |
|右移動       |D　→キー	            |方向キー右  |
|左移動       |A	　←キー	            |方向キー左  |
|下降         |W　↓キー	            |方向キー上  |
|攻撃         |左Ctrl  右Ctrl        | ボタン0  |
